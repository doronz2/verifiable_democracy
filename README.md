# Verifiable Democracy Lab

Open-source cryptographic tools for democratic processes people can verify for themselves.

Landing page for the Verifiable Democracy Lab, an initiative by [Doron Zarchy](https://doronzarchy.com/).

- Live (current): https://doronzarchy.com/democracy/
- Intended home: https://democracy.doronzarchy.com/ (served from this repo via GitHub Pages once DNS is configured)

## Projects

- [Aggios](https://github.com/doronz2/Aggios_implementation) — browser-based verifiable large-scale aggregation with proof generation and public verification
- [CRelections](https://github.com/doronz2/CRelections) — Rust implementation of coercion-resistant voting (JCJ/Civitas)
- [Subsenate](https://github.com/doronz2/Subsenate) — ZK-based sortition: private, unpredictable, publicly verifiable citizen selection
